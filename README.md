## 1. Diffusion models

This folder [DDPM–DDIM–ClassifierFree](./DDPM-DDIM-ClassifierFree) provides minimal, easy-to-read implementations** of key diffusion models, including **DDPM**, **DDIM**, and **conditional diffusion (classifier-free guidance)**.  
It contains both the source code and **pretrained model checkpoints**.
